# Documentation Guide

This folder contains the project documentation for the **International Bus Booking System (IBBS)**.
The files are formatted in **Markdown (.md)**, which is text-based. You can easily copy the content into Microsoft Word (.docx) or PowerPoint (.pptx).

## 1. Proposal Phase
*   **`Proposal_Document.md`**: Content for your project proposal Word document.
*   **`Proposal_Presentation.md`**: Content for your proposal defense slides.

## 2. Progress Phase
*   **`Progress_Document.md`**: Content for your progress report Word document.
*   **`Progress_Presentation.md`**: Content for your progress presentation slides.

## 3. Final Defense Phase
*   **`Final_Defense_Document.md`**: Content for your final project report (The "Thesis" / "Project Report").
*   **`Final_Defense_Presentation.md`**: Content for your final defense slides.

## How to use
1.  Open the `.md` file in VS Code or Notepad.
2.  Copy the text.
3.  Paste it into Word or PowerPoint.
4.  Format headings and bullet points as needed.
